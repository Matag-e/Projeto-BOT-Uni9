# Bem vindo ao BOT UNI9


O **BOT UNI9** foi desenvolvido com intenção de auxiliar em servidores do Discord com alguns coisas simples que normalmente não são tão explícitas no próprio programa e pensando também no usuario, programamos um BOT onde ele executa alguns [Comandos](https://github.com/Matag-e/SITEBOT/wiki/Comandos) que facilitará a vida do usuario.

O bot em si foi desenvolvido na linguagem python e projetado para funcionar dentro do aplicativo e site do Discord, onde ele foi criado e é identificado pelo **ID: 931767167718924288**, que foi gerado pela própria plataforma e tem licença para ser utilizado e é apresentado da seguinte forma:

![botdiscord](https://github.com/Matag-e/SITEBOT/assets/50279606/346ae3e8-e072-4462-9176-2a6a84e51bb3)

Após a criação e definição de nome e imagem do Bot, começa a etapa de programação, onde programamos as funcionalidades do bot, para que exista uma ligação do bot na plataforma e o código que desenvolvemos a plataforma gere um token, que é um código que realiza a ligação entre ambos, esse código deve ser protegido e confidencial, pois qualquer um que tiver esse código poderá realizar alterações no projeto.

## IDENTIFICADOR

O Bot funciona da seguinte forma, um usuário escreve um comando no chat do discord junto de um prefixo pré-programado que identifica a palavra como um comando, no nosso caso é o **“.”** e o bot. identificara se é um comando para qual ele foi programado para responder, se sim ele realizara uma ação, se não for ele irá ignorar.

## **COMO USAR OS [COMANDOS](https://github.com/Matag-e/SITEBOT/wiki/Comandos)**

Aqui temos, o usuário ( Zero ) usando o comando “.help” que é o comando que ensina como usar o bot:

![cmdhelp](https://github.com/Matag-e/SITEBOT/assets/50279606/7bfe467b-5227-456e-bb0c-7b98fac3ab31)

E como é um dos comandos aceitáveis resultará na seguinte mensagem:

![imgHelp](https://github.com/Matag-e/SITEBOT/assets/50279606/2111568f-40a8-4ea2-90c7-f7346f445fa1)

Cada comando irá realizar uma ação diferente pré-programada dentro da programação, caso alguém digite um comando que não foi informado no desenvolvimento, o bot passara a ignorar a mensagem e irá responder apenas aqueles para ele tem instruções pré-definidas.

## **IDENTIFICAÇÃO DOS USUARIOS**

![codigo](https://github.com/Matag-e/SITEBOT/assets/50279606/3ac0b6ad-47b0-43e2-8fe2-7486a97b213a)


No código nos declaramos 3 classes de usuários possíveis que são reconhecidos pelo bot e apenas essas 3 classes podem realizar ações que são as seguintes:

**Client** = usuários do chat por texto nas comunidades do discord:

![client](https://github.com/Matag-e/SITEBOT/assets/50279606/b93ef919-c356-4376-aad4-8cfbc9b6ecd6)

**Bot** = é a classe do próprio bot que reconhece ações feitas por ele mesmo:

![bot](https://github.com/Matag-e/SITEBOT/assets/50279606/9603b912-8104-4d0c-b5fe-78832778604b)

**Voice_Client** = são aqueles usuários que estão participando de ligações por voz dentro do servidor na qual o bot está presente:

![voiceclient](https://github.com/Matag-e/SITEBOT/assets/50279606/3077ecae-5172-4c62-ab08-48d938207fb8)
 
Declarando a classe dessa forma todos eles quando utilizarem o prefixo( . ) + comando o bot reconhecerá e irá responder. 

## **WEBSITE**
Durante o desenvolvimento do bot em si, surgiu a ideia de criarmos um site feito puramente em html e css que se adaptaria a tela do usuário, no qual serviria como principal meio de contato com os desenvolvedores para auxiliar os usuários caso aja duvidas ou o projeto apresente problemas técnicos ou mal funcionamento, e hoje atualmente ele está hospedado via Netlify e pode ser acessado [CLICANDO AQUI](https://bot-discordu9.netlify.app).

O bot foi programado para quando usarem “.link” ele irá mandar o endereço online do site feito para entrar em contato com os desenvolvedores e quando o link for aberto esta tela será aberta em seu navegador:

![imgSite](https://github.com/Matag-e/SITEBOT/assets/50279606/de8cf842-34ba-40cb-8180-5401c6faf401)

Apresentando uma mensagem de boas vindas e agradecendo por usar nosso bot, informamos os links de contato principal dos usuários que podem tirar duvidas sobre o projeto, usando Instagram, WhatsApp e o próprio discord como meios de contato com a equipe.
